## BUG-1: Button component missing forwardRef

**Time**: 22:48 | **Session**: admin-browser | **URL**: http://localhost:3001/admin/namespaces | **Severity**: minor

**Error**:
```
Warning: Function components cannot be given refs. Attempts to access this ref will fail. Did you mean to use React.forwardRef()?

Check the render method of `DialogClose`.
    at Button (./src/components/ui/button.tsx:41:11)
    at DialogClose (./node_modules/@base-ui/react/esm/dialog/close/DialogClose.js:25:13)
```

**Root Cause**: `Button` component is a plain function component without `React.forwardRef()` wrapper. `@base-ui/react`'s `DialogClose` passes a ref to its child `Button`, causing React warning.

**Fix**: Wrap `Button` component with `React.forwardRef()` in both user-frontend and admin-frontend.
